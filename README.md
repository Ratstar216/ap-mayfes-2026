# Applied Physics May Festival Website

Base website for the Applied Physics project at the UTokyo May Festival.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Motion
- Vercel

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Useful Scripts

```bash
npm run typecheck
npm run lint
npm run build
```
